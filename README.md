## 🛡️ Features

### Core Protection
- **Base64 Encoding** - Encode HTML with Base64 for basic obfuscation
- **iFrame Isolation** - Load HTML in isolated iframe for enhanced security
- **HTML Minification** - Remove whitespace and comments to reduce size

### Advanced Encryption
- **XOR Encryption** - Apply XOR cipher with random key
- **RC4 Encryption** - Apply RC4 stream cipher encryption
- **Multi-layer Encoding** - Apply multiple encoding layers for maximum protection
- **LZ Compression** - Compress output using LZ algorithm

### Anti-Debug & Anti-Inspect
- **F12 Blocking** - Prevent F12 key from opening DevTools
- **DevTools Blocking** - Block Ctrl+Shift+I keyboard shortcut
- **Right-click Blocking** - Disable right-click context menu
- **View Source Blocking** - Prevent viewing page source (Ctrl+U)
- **DevTools Detection** - Detect and break execution if DevTools is open
- **Anti-Tamper Protection** - Detect code modification attempts

### Access Control
- **Password Protection** - Require password to view content
- **Expiry Date** - Set expiration date for content
- **Domain Locking** - Restrict access to specific domains
- **Watermark Signature** - Embed invisible watermark in code

### User Experience
- **Pre-configured Templates** - Quick-start with security presets
- **Drag & Drop Upload** - Easy file input with drag and drop
- **Real-time Statistics** - Character count, line count, and size metrics
- **Export Options** - Download as HTML or export as ZIP with README
- **Live Preview** - Preview obfuscated output before downloading

## 🚀 Quick Start

1. **Open** the `index.html` file or [DIRECT LINK](https://redx-corporation.github.io/html-obfuscator-advanced/) in any modern web browser
2. **Choose** a security template or configure custom settings
3. **Input** your HTML via:
   - Direct paste in the textarea
   - File upload (drag & drop or click to browse)
4. **Click** "OBFUSCATE HTML" to process your code
5. **Download** or export your obfuscated HTML

## ⚠️ Important Notes

### Security Warning
Using **Access Control** or **Anti-Debug/Anti-Inspect** protections may reveal protection mechanisms in the obfuscated code. For maximum code hiding, it's recommended to re-obfuscate with only Basic Security (Base64 encoding and iFrame isolation).
